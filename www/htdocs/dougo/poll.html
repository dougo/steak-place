Chugchanga-L Poll 1999

Welcome to the 1999 Chugchanga-L Poll! Please use the applet below to submit your 20 favorite records of 1999. To run it, you'll need a browser capable of running Java 2 applets; it may ask you to download and install the Java 2 plugin. I hope the interface is fairly intuitive, but read the instructions anyway. See also the eligibility rules. The poll will close on February 13th.

If your browser is not displaying the applet above, you can download the JAR file (right-click and "Save Link As...", or something similar) and run it with the Java 2 interpreter directly. On Unix and Windows, you can do this from the command line with: 

        java -jar poll.jar
If this is not an option for you, or you'd rather not bother, you can just send me your list directly, along with any comments you'd like to appear with your list. Let me know if you don't want your name and/or email address made public. Please include the artist name, record name, and label name for each record. Have a look at the master list if you need to jog your memory.
