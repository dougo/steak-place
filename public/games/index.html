Games

I have invented several games:

  • Elementary, a deduction game for 3-6 players using an Aquarius deck
  • Intersect, a simultaneous-selection game for 2 players using Looney Pyramids and a few numbered cards
  • Pylon, an abstract strategy game for 2 players using Looney Pyramids and a 5x6 rectangular board
  • Quintazone, a tile-laying game for 2-5 players using Looney Pyramids and an Aquarius deck

I have also adapted the rules to Gnostica to be used with the Minimalist Gnostica Deck rather than a tarot deck.
