    • Structure and Interpretation of Computer Programs
    • R5RS, The Revised5 Report on the Algorithmic Language Scheme
    • Larceny, a Scheme implementation for Solaris written at Northeastern. Features a state-of-the-art generational garbage collector.

    • Kawa, a Scheme implementation written in Java that compiles to Java bytecodes and allows access to Java classes.
